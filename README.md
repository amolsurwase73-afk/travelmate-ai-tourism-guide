 TravelMate AI Tourism Guide

AI-powered tourism guide assistant built with React, TypeScript, Tailwind CSS and Google Gemini AI.

 Features

- AI Travel Planner
- Personalized Itinerary Generation
- Tourist Attractions Recommendations
- Local Food Guide
- Budget Calculator
- Travel Tips
- Smart Packing List
- AI Chat Assistant

 Tech Stack

- React.js
- TypeScript
- Tailwind CSS
- Google Gemini AI
- Node.js

 About Project

TravelMate AI helps travelers create personalized travel plans, discover destinations, explore local food, manage budgets, and get smart travel recommendations using Artificial Intelligence.

 Future Improvements

- Real-time Weather Integration
- Google Maps Integration
- Hotel Recommendations
- Voice Travel Assistant
